Fundamentals of HTML

HTML is the standard markup language for creating Web pages.

Structure & Elements

HTML consists of a series of elements, which you use to enclose or wrap, different parts of the content to make it appear a certain way, or act a certain way.

Element Description
HTML This is the root element of an HTML page. It wraps all the content on the entire page. It is sometimes known as the root element.
Head A container for all the head elements relating to metadata. These typically include title, link, meta, script, and base.
  • Title: Displays in the tab of the browser as the title of the page, and is userful for SEO purposes.
  • Link: Used to link to external style sheets.
  • Meta: Contains metadata about the HTML document, such as the character set, page description, keywords, author of the document, and more.
  • Script: Used to define client-side JavaScript.
  • Base: Used to specify the base URL and target for all relative URLs in a page.
Body Contains all the content of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
  • Header: Contains introductory content or navigational aids for the page.
  • Nav: Contains a collection of navigational links.
  • Main: Contains the main content of the page, such as articles, blog posts, etc.
  • Section: Contains related content grouped together, such as blog posts, articles, etc.
  • Article: Contains independent, self-contained content, such as blog posts, articles, etc.
  • Aside: Contains content that is tangentially related to the content around it, such as sidebars, pull quotes, etc.
  • Footer: Contains the footer of the page, such as copyright information, contact information, etc.
